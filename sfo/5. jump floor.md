## 题目描述

```
一只青蛙一次可以跳上1级台阶，也可以跳上2级。求该青蛙跳上一个n级的台阶总共有多少种跳法（先后次序不同算不同的结果）。
```

```cpp
#include <iostream>
using namespace std;

class Solution {
public:
    int jumpFloor(int number) {
        if (number == 1) {
            return 1;
        }

        if (number == 2) {
            return 2;
        }

        int a = 1;
        int b = 2;
        int c;

        for (int i = 3; i <= number; i++) {
            c = a + b;
            a = b;
            b = c;
        }

        return c;
    }
};

int main(int argc, char *argv[])
{
    Solution s;

    cout << s.jumpFloor(10) << endl;

    return 0;
}
```
